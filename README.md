# 📊 Customer Behavior Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow using real-world retail customer data. It covers the complete analytics process, from loading and cleaning raw data to performing exploratory data analysis (EDA), solving business problems using SQL, and building an interactive Power BI dashboard.

The objective is to transform raw business data into meaningful insights that support data-driven decision-making.

---

## Dataset

The dataset contains retail customer shopping information, including:

* Customer Details
* Age and Gender
* Product Category
* Purchase Amount
* Location
* Payment Method
* Shipping Type
* Discount Applied
* Subscription Status
* Previous Purchases
* Review Ratings
* Seasonal Trends

## How to Run

1. Clone this repository.
2. Install the required Python libraries:

   * pandas
   * numpy
   * matplotlib
   * seaborn
3. Open the Jupyter Notebook files and execute them in order.
4. Import the cleaned dataset into your preferred SQL database (PostgreSQL, MySQL, or SQL Server).
5. Run the SQL scripts to perform business analysis.
6. Open the Power BI `.pbix` file to explore the interactive dashboard.
7. Review the project report and presentation for a complete summary of the analysis.
